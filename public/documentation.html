Movie API Documentation

Request URL Description URL Parameters Request Body Response
GET /movies Returns a list of all movies. None None JSON array of film objects.
GET /movies/:title Returns data about a single movie by title. :title – a single movie by title (string). None JSON object with description, genre, director, image URL, featured.
GET /genres/:name Returns data about a genre by title. :title – the film genre by title (string). None JSON object with title and description.
GET /directors/:name Returns data about a director by name. :name – the director name (string) None Returns a JSON object with data about a director (bio, birth year, death year).
POST /users Allows a new user to register None JSON, e.g.: {"username": "camila", "email": "camila@example.com", "birthday": "1998-04-12"} Returns a JSON object with the created user data.
PUT /users/:username Updates existing user's info. :username – current username JSON (later), e.g.: { "Username": "newName", "Password": "newPassword", "Email": "newEmail@example.com", "Birthday": "1990-01-01" } JSON object with updated user data.
POST /users/:username/movies/:movieId Adds a film to the user's list of favorites. :username – user’s username
:movieId – movie ID 		None (for now). Confirmation message and updated user document (JSON).
DELETE /users/:username/movies/:movieId Removes a film from the user's list of favorites. :username – user’s username
:movieId – movie ID 		None Confirmation message and updated user document (JSON).
DELETE /users/:username Deregisters an existing user. :username – user’s username None Confirmation message that the user has been removed.
