Film API Documentation

Request URL Description URL Parameters Request Body Response
GET /films Returns a list of all films. None None Plain text (placeholder): "This will return a list of all films". Later: JSON array of film objects.
GET /films/:title Returns data about a single film by title. :title – a single film by title (string). None Plain text (placeholder). Later: JSON object with description, genre, director, image URL, featured.
GET /genres/:title Returns data about a genre by title. :title – the film genre by title (string). None Plain text (placeholder). Later: JSON object with title and description.
GET /directors/:name Returns data about a director by name. :name – the director name (string) None Plain text (placeholder). Later: Returns a JSON object with data about a director (bio, birth year, death year).
POST /users Allows a new user to register None JSON (later), e.g.: {"username": "camila", "email": "camila@example.com"} Plain text (placeholder). Later: returns a JSON object with the created user data.
PUT /users/:username Updates existing user's info. :username – current username JSON (later), e.g.: {"username": "newName"} Plain text (placeholder). Later: JSON object with updated user data.
POST /users/:username/films/:filmId Adds a film to the user's list of favorites. :username – user’s username
:movieId – movie ID 		None (for now). Plain text (placeholder). Later: confirmation message or updated user data.
DELETE /users/:username/films/:filmId Removes a film from the user's list of favorites. :username – user’s username
:movieId – movie ID 		None Plain text (placeholder). Later: confirmation message.
DELETE /users/:username Deregisters an existing user. :username – user’s username None Plain text (placeholder). Later: confirmation message that the user has been removed.
